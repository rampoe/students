селекторы (selectors) - выбирают элементы из HTML (select = выбирать)

для элементов: h1, h2, p, a
для классов: .text-red, .background-blue
для идентификаторов (id): #main-heading, #theFirstPage

отличие между классами и идентификаторами в том, что классы могут использоваться многократно,
а идентификаторы - только один раз

в целом, для CSS лучше использовать классы, чем id
